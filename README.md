Aplikasi Mobile Asesmen Informatika SMP merupakan aplikasi pembelajaran berbasis mobile yang dikembangkan menggunakan Flutter dan Firebase. Aplikasi ini dirancang untuk mendukung proses asesmen pembelajaran dengan mengintegrasikan konsep Assessment for Learning (AfL) dan Assessment as Learning (AaL) dalam pendekatan deep learning pedagogy.

Melalui aplikasi ini, siswa tidak hanya diminta memilih jawaban benar atau salah, tetapi juga memilih alasan yang sesuai dengan konteks soal. Setelah menyelesaikan asesmen, siswa memperoleh umpan balik hasil belajar, mengisi jurnal refleksi, serta menyimpan riwayat asesmen dalam bentuk portofolio digital.
